# Screenshots

Drop PNG screenshots in this folder using the exact filenames below. They are
referenced from the main [README](../../README.md) and will render automatically.

Recommended width: ~1200px. Use placeholder/sample data — **don't include real
players' names or passcodes** in the screenshots you publish.

| Filename | What to capture |
| --- | --- |
| `predictions.png` | The **🔮 Predictions** tab with a couple of open matches showing the Home/Draw/Away buttons and a countdown timer. |
| `results.png` | The **📅 Past Results** tab showing a resolved match with revealed votes and points awarded. |
| `leaderboard.png` | The **🏆 Leaderboard** tab with a few ranked players. |
| `rules.png` | The **📝 Points Rules** tab. |
| `admin.png` | The **⚙️ Admin Panel** (unlocked) showing player management and the add-match form. |
| `login.png` | The passcode **login modal** ("Enter Your Arena Passcode"). |

To capture them: run the app locally (`npm start`), log in with the default
admin (`ADMN`), add a few sample players/matches, then screenshot each tab.
